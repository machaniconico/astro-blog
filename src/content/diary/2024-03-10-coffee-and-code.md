## 今日やったこと

午前中はタスク管理アプリのフロントエンドを触っていた。
Reactのコンポーネントを整理して、状態管理を少しリファクタリング。

```javascript
// こんな感じでカスタムフックを整理した
function useTasks() {
  const [tasks, setTasks] = useState([]);

  const addTask = (title) => {
    setTasks(prev => [...prev, { id: Date.now(), title, done: false }]);
  };

  return { tasks, addTask };
}
```

午後はドキュメントを書いたり、設計について考えたり。
コードを書くのと同じくらい、設計を考える時間も好きだと気づいた。
